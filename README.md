# EV-Vehicle-Demand
This repository contains code for ML model which can provide insights on rising demands of EV vehicles and predict the growth of EV adoption in the future by making predictions using regression model. The project was done as part of internship program in collaboration with Edunet Foundation for Shell plc on AICTE.

The Jupyter Notebook file containing the project code can be accessed at [EV_Demand_Prediciton](https://nbviewer.org/github/Smasherrio/EV-Vehicle-Demand/blob/main/EV_Demand.ipynb#)
